# hwpx-image-compressor
HWPX 파일 이미지 압축 도구

hwpx 파일에 각종 이미지를 넣다 보면 어느 순간 용량이 엄청나게 커질 때가 있습니다.
그림파일이 한두개라면 상관없지만 수십, 수백개라면 이 한글파일의 용량을 어떻게 줄일까요?
이 코드를 실행하고 hwpx파일을 업로드 하세요. 그럼 이미지파일 크기만 줄여줄 겁니다.

## 주의사항
- 이 코드는 바이브코딩을 통해 작성된 실험적인 프로젝트입니다.  
- 중요한 문서는 반드시 **백업 파일**을 만들어 둔 뒤 사용하세요.  
- 이 프로그램 사용으로 인해 발생하는 데이터 손실, 파일 손상 등에 대해 작성자는 어떠한 책임도 지지 않습니다.  
- 버그 제보나 개선 아이디어는 언제든지 이슈로 남겨 주세요.
